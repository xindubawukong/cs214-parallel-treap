
Parallel Set Operations on Treaps

### Download

```
git clone --recurse-submodules git@github.com:xindubawukong/cs214-parallel-treap.git
```

### Build and Run

```
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./main
```

You may need to add a line `#include "parlay/parallel.h"` into `PAM/include/pam/utils.h`.
